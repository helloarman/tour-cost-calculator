# Tour Contribution Calculator

A **simple and professional web app** to manage and calculate tour trip expenses among members.  
Built with **HTML, CSS, and JavaScript**, with local storage support for caching data.

## Features

- Add or remove **tour members**.  
- Track individual **contributions** (title + amount).  
- Calculate **total contribution** and **equal share per member**.  
- Display **due amount** for each member to balance the expenses.  
- **Reset** all data for a new trip.  
- **Local storage** caching to persist data.  
- Modern and clean **UI with responsive design**.  
- Professional icons for delete actions and clean card layout.
